Scherzo.js by byee01

An evil library for pranking.

A quick project during the Facebook Hackathon at CMU to teach myself some JavaScript/jQuery and code management.

Scherzo ("trick" in Italian) is a jQuery plugin for pranking people, just in time for April fools. Includes goodies like Tourettes form input, blurry text and runaway links! I did this in 8 hours so if you have any improvements it would be much appreciated.

See it in action here!

Dependencies

jQuery - tested with 1.4.x

Install

On load, fire up Scherzo. There are defaults, but you can check the source to tweak options.
